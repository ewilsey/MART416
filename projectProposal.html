 Web Development and Animation Proposal 


      
Abstract


        The goal of this project is to showcase a proficiency in web development
      to potential employers after graduation. This website will be created in
      Atom and maintained by me throughout the semester.

      
Introduction


        Web Development is an artform, it can be functional as well as formal.
      The site itself is art, and what it is designed to display is more art. I
      envision this project website to eventually become a place where artists
      can come to collaborate, store, and showcase their art with me.

      
Description


        My website and all of the content contained within during this semester
      are my sole creations. The style and organization will change over time.
      The goal for this semester is for this webisite to serve as the home for
      all of my future projects. The success in this project and in my Web
      Server Tech class will determine if I can implement this website to be
      used for a community of artists, not just myself.

      
Significance


        This website will showcase my abilities as a web developer and designer.
      The contents held within will showcase my abilities as an artist,
      animator, editor, sound designer and script writer.
