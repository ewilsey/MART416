Web Page Design



Progress

I set out to create an animation and a website. My main focus for this semester
was on website design because it is my strongest area per interest and ability.
The Website has gone through many redesigns. I started with a list of html links.
All of these links had to work, and all of internal links had to contain content
linked to my CSS and JS. With working links and content behind these links, the
next step was creating a visually appealing way to display them. I started
with a navigation bar across the top but it was not re-size friendly - the straight
and appealing bar displayed on PC became a stack of uneven bars when displayed
on a cellphone. The resolution to this re-sizing conundrum came in the form the
almighty hamburger menu. It worked so well I made its background a tomato.
